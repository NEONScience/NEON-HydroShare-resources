# NEON-HydroShare-resources
repository to create and maintain NEON site-level resources published in CUAHSI HydroShare

## Navigating the repository

[/exploration-scripts](https://github.com/NEONScience/NEON-HydroShare-resources/tree/main/exploration-scripts) - helper scripts coded in both R and Python that allow users to better interpret and explore the JSON file included in NEON site-level resources published in HydroShare.

[/resource-metadata](https://github.com/NEONScience/NEON-HydroShare-resources/tree/main/resource-metadata) - Python scripts that use HydroShare's [hsclient](https://github.com/hydroshare/hsclient) to create and modify NEON site-level resources published in HydroShare.
